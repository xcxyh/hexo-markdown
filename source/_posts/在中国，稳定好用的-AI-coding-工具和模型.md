## Models：

Google 系列模型：
Gemini-3-pro
Gemini-3-flash

怎么才能获取到 Gemini 最新模型的 API 调用权限呢，一般这个是最让大家头疼的问题。这里可以参考方案： https://blog.csdn.net/m0_63251896/article/details/156537288，白嫖价值 300美元的 Gemini API 调用额度。有效期长达90天。

![image-20260111170105189](https://raw.githubusercontent.com/xcxyh/xcxyh.github.io/image-save/images/2024202601111701573.png)

智谱AI 系列模型：
GLM-4.7
GLM-4.6

智谱系列模型 应该是国内最好用的 ai coding 模型了，而且官网提供了非常详细的接入教程，可以方便的集成 GLM 模型到 Claude code、Cline、Cursor 等 AI Coding 工具中。算是非常适合Ai Coding 的国产平替了。这里也有多种付费套餐可选，而且新注册用户也会赠送大量tokens。这里的 glm-4.5-air 有 1500w tokens，可以做一些简单的任务，作为 Claude haiku 的平替。

![image-20260111165530430](https://raw.githubusercontent.com/xcxyh/xcxyh.github.io/image-save/images/2024202601111655711.png)

还可以购买 1000w tokens 3个月的glm 4.7套餐，个人开发基本够用了。其他的套餐可以自行探索，丰俭由人了。

![image-20260111170235751](https://raw.githubusercontent.com/xcxyh/xcxyh.github.io/image-save/images/2024202601111702016.png)


## Agents:

#### Claude Code + GLM-4.7

Claude Code + GLM 4.7 应该是体验非常好的国产平替了，推荐~

Claude Code 应该是目前最好用的 AI Coding Agent，有很多丰富的功能，mcps，skills、slash commands、hooks 等等，可以下载插件：Claude Code for VS Code ，方便查看和探索这些丰富的功能。

![image-20260111170523140](https://raw.githubusercontent.com/xcxyh/xcxyh.github.io/image-save/images/2024202601111705790.png)

#### Cline + Gemini3

Cline 作为 开源Ai Agent 方案，目前支持自定义配置所有模型，灵活性也是最大的，这里我选择配合 Gemini3 使用，效果不输cursor了。

![image-20260111171041260](https://raw.githubusercontent.com/xcxyh/xcxyh.github.io/image-save/images/2024202601111710605.png)

Cline 目前也有很多丰富的配置：例如 多仓库、 hooks、skills、yolo mode（打开后会自动执行任何命令，挺危险的配置），也是尽量跟上并对齐了目前业界最热门的功能了。

![image-20260111171149393](https://raw.githubusercontent.com/xcxyh/xcxyh.github.io/image-save/images/2024202601111711656.png)

整体体验下来，真的是太适合我们这些开不起 cursor pro 会员的开发者了~哈哈哈哈。

#### Cursor （免费额度太少，想要长期使用，需要开Pro会员）

Cursor 目前想使用 agent模式 + 自定义模型，需要开 pro 会员。但是 Cursor 还是目前体验最好的AI coding工具，哪怕是一些很一般的模型，Cursor Agent 模式的效果也能达到一个很不错的水平。怀念当时的cursor，白嫖了不少tokens。

![image-20260111171757583](https://raw.githubusercontent.com/xcxyh/xcxyh.github.io/image-save/images/2024202601111717044.png)

目前来说，我用的最多的就是 Cline + Gemini3 和 Claude Code + GLM-4.7 这两个组合了。体验虽然没有 cursor 和 claude 最新模型那样好的体验，但是做个人项目开发来说已经足够了，足够在这些ai 工具上验证自己的想法了。
