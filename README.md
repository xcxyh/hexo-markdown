# hexo-markdown

个人 hexo 博客的源码仓库，基于 Hexo 静态站点生成器构建。

## 项目介绍
这是一个使用 Hexo 搭建的个人博客项目，用于分享技术文章、生活感悟等内容。通过 Markdown 编写文章，借助 Hexo 的高效渲染能力生成静态 HTML 页面，支持主题定制、插件扩展和快速部署。

## 核心功能
- **Markdown 写作**：使用简洁的 Markdown 格式创作内容，降低排版成本。
- **静态站点生成**：Hexo 快速将 Markdown 文件转换为静态 HTML，提升页面加载速度。
- **主题与插件**：支持更换 Hexo 主题（如 Next、Fluid 等），通过插件扩展功能（如评论、统计等）。
- **便捷部署**：可通过 `hexo deploy` 命令快速部署到 GitHub Pages、Netlify 等平台。

## 技术栈
- 静态站点生成器：Hexo
- 内容格式：Markdown
- 版本控制：Git
- 运行环境：Node.js

## 目标用户
适用于希望拥有个人独立博客、喜欢用 Markdown 写作、追求站点性能和自定义化的用户。
